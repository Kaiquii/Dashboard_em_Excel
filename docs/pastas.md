# Pastas das planilhas

Use assim:

```text
inputs/      = planilhas originais/base pendentes, que ainda vamos transformar em dashboard
referencias/ = planilhas prontas/corretas para usar como referencia
outputs/     = planilhas geradas pelos scripts
```

As planilhas que ainda vamos mexer ficam sempre dentro de `inputs/`.

Quando uma planilha ja estiver pronta/correta, ela deve sair de `inputs/` e ficar somente em `referencias/`.

Exemplo:

```text
inputs/
  CQ_Materia_Prima_Fornecedores_2026/
    CQ Matéria Prima - Fornecedores 2026.xlsm
```
